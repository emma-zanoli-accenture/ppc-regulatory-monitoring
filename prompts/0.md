1. Scaffold a Vite React-TS project in the current directory.
2. Install and configure Tailwind CSS (postcss + autoprefixer), wire it into the global stylesheet, and confirm utility classes work.

   - components/   (reusable UI: buttons, cards, badges, modals, tables)

   - pages/        (route-level screens)
   - data/         (all mock/seed data lives here, typed)
   - lib/          (helpers: formatting, fake-AI utilities, delay)
   - types/        (shared TypeScript types)
